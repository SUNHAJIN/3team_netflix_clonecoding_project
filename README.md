## 기능 📸
* [랜딩페이지] `설날에 뭐보지?` 소개
* [로그인] 일반 로그인, 소셜 로그인(네이버, 카카오, 페이스북)
* [회원가입] 쉽고 간결한 회원가입
* [장르선택] 유저의 선호 영화를 선택
* [메인페이지] 5개의 추천 영화 리스트(선호영화 유사도, 평점높게줄영화, 장르, 감독, 설기간), 영화 상세 정보 확인
* [음성검색] 음성으로 명대사 검색 시, 해당 영화 리스트 확인
